# 💈 MR you - Premium Barber Shop

A high-end, multi-lingual barber shop management and booking application. Built with React, Tailwind CSS, and Firebase.

## 🚀 Features

- **Real-time Booking:** Clients can book appointments with specific barbers.
- **Multi-lingual Support:** Available in English, French, and Arabic (RTL support).
- **Role-based Access:**
  - **Admin/Manager:** Full control over barbers, bookings, and shop settings.
  - **Worker:** Manage personal status and view assigned bookings.
  - **Client:** View barber availability and book appointments.
- **Automated Management:** Automatically marks bookings as "missed" if clients are more than 10 minutes late.
- **Notifications:** Real-time alerts for barbers when new bookings are made.
- **Custom Branding:** Upload your own shop logo and barber photos.

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite, Tailwind CSS 4
- **Database:** Firebase Firestore
- **Animations:** Motion (Framer Motion)
- **Icons:** Lucide React

## 📦 Deployment to Netlify

This project is pre-configured for Netlify.
1. Connect your GitHub repo to Netlify.
2. Build Command: `npm run build`
3. Publish Directory: `dist`

## ⚙️ Setup

1. Clone the repository.
2. Install dependencies: `npm install`
3. Configure your Firebase credentials in `src/firebase-applet-config.json`.
4. Run locally: `npm run dev`
